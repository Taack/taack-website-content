== Exemple de code
[source,groovy]
.Exemple de code de kanban 1.
----
UiKanbanSpecifier buildTodoKanban(List<Todo> todos) {
    new UiKanbanSpecifier().ui {                                                                   <1>
        column 'HeaderStr', Style.ALIGN_CENTER, Gantt2Controller.&changeTodoProgression as MethodClosure, [newProgression: 0], {    <2>
            custom "Custom html code"                                                              <3>
            todos.forEach { Todo todo ->
                card todo, Gantt2Controller.&showTodo as MethodeClosure, todo.id, {                <4>
                    cardField todo.what_                                                           <5>
                    cardFieldRaw "Custom html code"                                                <6>
                    cardAction ActionIcon.SHOW * IconStyle.SCALE_DOWN, Gantt2Controller.&doSomething as MethodClosure, todo.id   <7>
                }
            }
        }
    }
}

----

[source,groovy]
.Écrit sur un service pour enregistrer le menu contextuel
----
@PostConstruct
void init() {
    TaackUiService.registerContextualMenuClosure(Todo, new UiMenuSpecifier().ui {               <8>
        menu 'Show Gantt', Gantt2Controller.&showGanttByTodo as MethodClosure, [id: this.params['id']]    <9>
    }
}
----

<1> Créer le kanban
<2> Créer une colonne et un en-tête de colonne, permettre de passer une action et des paramètres comme kbd:[Drag+Drop] pour la colonne
<3> Permettre d'ajouter le code HTML personnalisé pour le corps de la colonne
<4> Créer des cartes dans la colonne, permettre de passer an action and params une action et des paramètres comme kbd:[MouseDoubleClic] pour la carte
<5> Créer une ligne avec le champ dans la carte, noter le *tiret bas* à la fin du nom du champ
<6> Permettre d'ajouter le code HTML personnalisé
<7> Permettre d'ajouter une icône avec une action liée au clic
<8> Enregistrer le menu contextuel comme kbd:[MouseRight] pour la carte
<9> Créer un élément de menu avec une action
